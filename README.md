# Hotel Reservation Manager

Αυτός ο φάκελος περιέχει μια ολοκληρωμένη HTML/CSS/JavaScript εφαρμογή για διαχείριση κρατήσεων ξενοδοχείου.

## Άνοιγμα

1. Αποσυμπίεσε το ZIP.
2. Άνοιξε το `index.html` με Chrome, Edge ή Safari.
3. Τα δεδομένα αποθηκεύονται τοπικά στον browser με `localStorage`.

Για καλύτερη εμπειρία PWA/offline cache, άνοιξέ το μέσα από έναν μικρό server, π.χ. VS Code Live Server. Το service worker δεν ενεργοποιείται όταν το αρχείο ανοίγει απευθείας ως `file://`, αλλά η εφαρμογή λειτουργεί κανονικά.

## Περιλαμβάνει

- Dashboard με πληρότητα, ελεύθερα δωμάτια, αφίξεις, αναχωρήσεις, έσοδα και υπόλοιπα.
- Ημερολόγιο τύπου tape chart ανά δωμάτιο και ημερομηνία.
- Έλεγχο διαθεσιμότητας ανά ημερομηνίες, τύπο δωματίου και άτομα.
- CRUD κρατήσεων με αυτόματο μπλοκάρισμα διπλοκράτησης.
- Μητρώο δωματίων με τύπο, τιμή, χωρητικότητα και λειτουργική κατάσταση.
- Housekeeping / καθαριότητα.
- Πληρωμές, προκαταβολές, επιστροφές και υπόλοιπα.
- Μηνιαίες αναφορές με πληρότητα, έσοδα, ADR και RevPAR.
- JSON backup/import και CSV export αναφορών.

## Σημαντικό

Η εφαρμογή είναι local-first. Για κανονική χρήση από πολλές συσκευές ή πολλούς χρήστες χρειάζεται backend/database, π.χ. Supabase ή Firebase.
